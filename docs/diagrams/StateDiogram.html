Диаграмма состояний агента
📌 Состояния агента: UNINITIALIZED → INITIALIZING → REGISTERING → IDLE → FETCHING_TASK → EXECUTING → SENDING_RESULT → IDLE (цикл). При ошибках переход в ERROR с возможностью восстановления. При сигнале завершения (SIGTERM) - переход в SHUTDOWN.

🎯 Ключевые переходы: успешная регистрация → IDLE, получение задачи → EXECUTING, выполнение завершено → SENDING_RESULT, результат отправлен → IDLE.
